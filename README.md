# Pokémon API Integration

This project integrates with the Pokémon API to fetch and display details of Pokémon. It provides a simple interface to interact with the data, showcasing information such as Pokémon abilities, stats, and types.

## Features

- Fetch and display Pokémon information from the Pokémon API.
- Display detailed stats for each Pokémon.
- Search for Pokémon by name.
- Responsive layout to work across devices.

## Tech Stack

- **Node.js**: JavaScript runtime used for the back-end.
- **Express.js**: Web framework to handle routes and server-side logic.
- **Pokémon API**: Public API to fetch Pokémon data.
- **EJS**: Template engine to render views.
- **Tailwind CSS**: CSS framework for styling the front-end.

## Prerequisites

Before you begin, ensure you have the following installed on your system:

- **Node.js**: [Download and install Node.js](https://nodejs.org/)
- **npm**: Comes with Node.js
